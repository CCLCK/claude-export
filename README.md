# Claude Export

一个用于导出 `claude.ai` 对话的 Chrome 扩展。

它会把聊天记录导出为本地 HTML，并尽量把正文资源、控件、附件预览和 Obsidian 配套 Markdown 一起整理好，方便离线查看、归档和检索。

## 功能

- 导出对话为离线 HTML
- 导出配套 Obsidian Markdown 索引笔记
- 支持控件目录与问题目录导航
- thinking 导出可选
- 尽量内联正文资源与可预览附件

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions/`
2. 开启“开发者模式”
3. 选择“加载已解压的扩展程序”
4. 选中本项目目录

## 使用

1. 打开 `claude.ai` 对话页面
2. 点击扩展图标
3. 选择是否包含 thinking
4. 点击导出
5. 浏览器会下载：
   - 同名 HTML
   - 同名 Obsidian Markdown 模板

## Obsidian 配套

导出的 Markdown 主要用于：

- 归档
- 标签管理
- 双链管理
- 汇总用户问题

交互内容仍以 HTML 为准。
